# 技能名: 文件搜索

## 目标
根据用户描述的条件（名称、类型、日期、大小）搜索文件，返回按相关度排序的结果列表。支持按文件扩展名（如 PDF、Word、Excel、图片、视频）筛选。

## 前置工具
- `find_files`
- `scan_files`

## 执行步骤
1. 从用户输入中提取搜索条件：文件名关键词、扩展名、日期范围、大小范围
2. 调用 `find_files` 执行搜索，设置合理的 max_results
3. 如果结果为空，尝试放宽条件（去掉日期限制、模糊匹配名称）再搜一次
4. 将结果按时间线或分类组织展示给用户
5. 如果用户需要进一步操作（移动、删除），提供操作建议

## 陷阱与检查点
- 搜索路径不要从根目录开始，先确认用户要搜的目录
- 结果超过 50 个时分页展示，不要一次性输出
- 搜索隐藏文件需要用户明确要求
- 注意跨平台路径差异（Windows 用反斜杠，Linux 用正斜杠）
